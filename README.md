# 🌾 Crop Recommendation System

A Machine Learning web application that recommends the most suitable crops to grow based on soil and environmental conditions.



## 📊 Features
- **High Accuracy:** Built using a trained classifier with a high confidence rate.
- **7 Parameters:** Considers Nitrogen, Phosphorous, Potassium, Temperature, Humidity, pH, and Rainfall.
- **Interactive UI:** Simple web interface built with Streamlit.

## 🛠️ Tech Stack
- **Python** (Data Processing & ML)
- **Scikit-Learn** (Random Forest/GaussianNB Classifier)
- **Streamlit** (Web Framework)
- **Pickle** (Model Serialization)

## 📁 Project Structure
- `app.py`: Main web application script.
- `model.pkl`: Trained Machine Learning model.
- `scaler.pkl`: Scaler for feature normalization.
- `label_encoder.pkl`: Encoder for crop names.
- `requirements.txt`: List of required libraries.
